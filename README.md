# ronin-shop-sf

## Features

- **User Management**: Register new users, login, logout, edit user profiles, and manage roles (admin).
- **Admin Dashboard**: Admin users can view and manage users' roles.
- **Security**: Password hashing, login authentication, and role-based access control.
- **Customer Profile**: Users can create, edit, and manage their customer profiles, including email and password updates.
- **Product Management**: (CRUD) create, edit, and delete products; upload images; manage price and stock; automatic SEO-friendly slug generation with redirect handling; product search and filtering.
- **Category Management**: (CRUD) create, edit, delete categories; search functionality; deletion restricted to empty categories.
- **Catalog**: Product listing with search, price filtering, and sorting to improve product discoverability.
